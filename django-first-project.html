Cómo Configurar Django y Crear tu Primer Proyecto

Publicado el 22/03/2023
Leodanis Pozo Ramos

Django is a popular Python-based web framework used for building dynamic web applications. In this article, we will guide you through the process of setting up Django and creating a first project.

Prerequisites

Before we begin, make sure you have the following prerequisites installed on your computer:

  • Python (version 3.6 or above)
  • pip (Python package manager)
  • virtual environment (recommended)

Setting up Django

There are different ways to set up Django, but we recommend using a virtual environment to avoid conflicts with other Python packages on your computer. Here's how to set up Django using a virtual environment:

  1. Open your terminal or command prompt and create a new directory for your project. For example, mkdir myproject to create a new directory called "myproject".
  2. Navigate to the newly created directory using cd myproject.
  3. Create a new virtual environment using python -m venv env. This will create a new directory called "env" with the virtual environment files.
  4. Activate the virtual environment using source env/bin/activate on Unix-based systems or env\Scripts\activate.bat on Windows.
  5. Install Django using pip by running pip install django.

Creating a Django Project

Now that we have Django installed, we can create our first Django project:

  1. Run the command django-admin startproject myproject. This will create a new Django project with the name "myproject".
  2. Navigate to the newly created project directory using cd myproject.
  3. Start the development server by running python manage.py runserver. This will start the server on the default port 8000.
  4. Open your web browser and go to http://localhost:8000. You should see the default Django welcome page.

Creating a Django app

A Django project consists of one or more apps. Let's create a new app within our project:

  1. Run the command python manage.py startapp myapp. This will create a new Django app with the name "myapp".
  2. Open the views.py file within the "myapp" directory and add a simple view:
from django.http import HttpResponse

def hello(request):
    return HttpResponse("Hello, world!")
  1. Create a urls.py file within the "myapp" directory and add the following code:
from django.urls import path
from . import views

urlpatterns = [
    path('hello/', views.hello, name='hello'),
]
  1. Open the urls.py file within the project directory and add the following code:
from django.contrib import admin
from django.urls import include, path

urlpatterns = [
    path('admin/', admin.site.urls),
    path('myapp/', include('myapp.urls')),
]
  1. Restart the development server by stopping it with Ctrl+C and running python manage.py runserver again.
  2. Open your web browser and go to http://localhost:8000/myapp/hello/. You should see the message "Hello, world!".

Conclusion

You have successfully set up Django and created a simple project with an app. From here, you can start building more complex applications using Django's powerful features. To learn more about Django, check out the official documentation and various online resources.
