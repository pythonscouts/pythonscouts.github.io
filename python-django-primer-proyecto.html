Qué es Python y cuáles son sus características fundamentales?

Publicado el 22/03/2023
Leodanis Pozo Ramos

Django es un framework web popular basado en Python que se utiliza para crear aplicaciones web dinámicas. En este artículo, lo guiaremos a través del proceso de configuración de Django y la creación de un primer proyecto.

Requisitos previos

Antes de comenzar, asegúrese de tener los siguientes requisitos previos instalados en su computadora:

  • Python (versión 3.6 o superior)
  • pip (gestor de paquetes de Python)
  • entorno virtual (recomendado)

Configurando Django

Hay diferentes formas de configurar Django, pero recomendamos usar un entorno virtual para evitar conflictos con otros paquetes de Python en su computadora. Aquí se explica cómo configurar Django utilizando un entorno virtual:

  1. Abra su terminal o símbolo del sistema y cree un nuevo directorio para su proyecto. Por ejemplo, mkdir myproject para crear un nuevo directorio llamado "myproject".
  2. Navegue al directorio recién creado usando cd myproject.
  3. Cree un nuevo entorno virtual usando python -m venv env. Esto creará un nuevo directorio llamado "env" con los archivos del entorno virtual.
  4. Active el entorno virtual usando fuente env/bin/activate en sistemas basados en Unix o env\Scripts\activate.bat en Windows.
  5. Instale Django usando pip ejecutando pip install django.

Creando un Proyecto Django

Ahora que tenemos instalado Django, podemos crear nuestro primer proyecto Django:

  1. Ejecute el comando django-admin startproject myproject. Esto creará un nuevo proyecto Django con el nombre "myproject".
  2. Navegue al directorio del proyecto recién creado usando cd myproject.
  3. Inicie el servidor de desarrollo ejecutando python manage.py runserver. Esto iniciará el servidor en el puerto predeterminado 8000.
  4. Abra su navegador web y vaya a http://localhost:8000. Debería ver la página de bienvenida predeterminada de Django.

Creando una aplicación Django

Un proyecto de Django consta de una o más aplicaciones. Vamos a crear una nueva aplicación dentro de nuestro proyecto:

  1. Ejecute el comando python manage.py startapp myapp. Esto creará una nueva aplicación Django con el nombre "myapp".
  2. Abra el archivo views.py dentro del directorio "myapp" y agregue una vista simple:
from django.http import HttpResponse

def hello(request):
    return HttpResponse("Hello, world!")
  1. Cree un archivo urls.py dentro del directorio "myapp" y agregue el siguiente código:
from django.urls import path
from . import views

urlpatterns = [
    path('hello/', views.hello, name='hello'),
]
  1. Abra el archivo urls.py dentro del directorio del proyecto y agregue el siguiente código:
from django.contrib import admin
from django.urls import include, path

urlpatterns = [
    path('admin/', admin.site.urls),
    path('myapp/', include('myapp.urls')),
]
  1. Reinicie el servidor de desarrollo deteniéndolo con Ctrl+C y ejecutando python manage.py runserver nuevamente.
  2. Abra su navegador web y vaya a http://localhost:8000/myapp/hello/. Debería ver el mensaje "¡Hola, mundo!".

Conclusiones

Configuró Django con éxito y creó un proyecto simple con una aplicación. Desde aquí, puede comenzar a crear aplicaciones más complejas utilizando las potentes funciones de Django. Para obtener más información sobre Django, consulte la documentación oficial y varios recursos en línea.
